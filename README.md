# infiniteCanvas
可多人合作的(近乎)无限尺寸画布



#### 开发中...

#### 请注意,最新版本并不能正常运行,如果想尝试可下载[releases](https://github.com/xiyuesaves/infiniteCanvas/releases)中的版本



##### 部署方法

`npm i`

`npm run dev`

目前并非完成品,还有诸多问题,欢迎提交issues
